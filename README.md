# Timber Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Timber](https://img.shields.io/badge/Library-Timber-brightgreen.svg)](https://github.com/JakeWharton/timber)

Тестовое приложение для изучения библиотеки логирования Timber в Android. Демонстрирует различные уровни логирования и настройку приложения.

## Функциональность

- Кнопки для генерации логов разных уровней
- Инициализация Timber в классе Application
- Поддержка всех уровней логирования (Verbose, Debug, Info, Error)
- Возможность настройки для debug/release сборок

## Технологии

- Kotlin
- Timber
- ViewBinding
- Application class
